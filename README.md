# Overview ğŸ“˜

This repository contains a complete predictive maintenance analysis workflow built using Python, combining sensor analytics, anomoly detection, FMEA-driven reliability insights, and automated maintenance
scheduling. 

The project demonstrates how real-world maintenance engineering, failure mode assessment, and data science techniques can be integrated into a repeatable, scalable process.

This is a showcase of skills on:

    Data Analytics
    
    Statistical modeling
    
    Reliability engineering
    
    Failure prediction
    
    Maintenance planning
    
    Technical reporting
    
    Visualization (dashboards + gantt charts)
